# Maintenance System

Smart Tech Solutions – Service & Maintenance Management System.

A web-based platform designed to manage service requests, technicians, and maintenance operations efficiently.

## Features
- User registration and login
- Submit service requests
- Assign tasks to technicians
- Track job status

## Technologies
- Frontend: React (Vite)
- Backend: ASP.NET Core
- Database: SQL Server

## Run Project
npm install
npm run dev
